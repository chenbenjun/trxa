🎯 快速开始（推荐 Vercel）

  1. 访问 vercel.com 并注册
  2. 点击 "New Project" -> "Import Git Repository"
  3. 选择 chenbenjun/trxa 仓库
  4. 点击 "Deploy"，等待1-3分钟
  5. 获得HTTPS链接，复制到华为平板
  6. 按照使用指南安装到主屏幕
🚀 立即部署到 Vercel
