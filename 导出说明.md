## 🚀 使用方法

### 方式一：双击打开（最简单）

1. 解压压缩包
2. 找到 `index.html` 文件
3. 双击打开

**⚠️ 注意：** 直接打开可能因浏览器安全策略导致部分功能受限。

### 方式二：本地服务器（推荐）

#### Windows
```cmd
# 使用 Python（推荐）
python -m http.server 5000

# 使用 Node.js
npx serve .
```

#### macOS/Linux
```bash
# 使用 Python
python3 -m http.server 5000

# 使用 Node.js
npx serve .
```

然后在浏览器访问：http://localhost:5000

### 方式三：部署到静态服务器

可以将文件上传到：
- Nginx
- Apache
- 阿里云OSS
- 腾讯云COS
- GitHub Pages
- 任意支持静态网站的托管服务

### 浏览器要求
- 推荐使用 Chrome、Safari、Edge
- 需要启用 JavaScript
- 需要允许 LocalStorage
- 需要允许弹窗（用于打印）

建议定期备份数据，在浏览器控制台执行：

### 存储限制
- LocalStorage 限制 5-10MB
- 建议使用压缩后的图片
- 定期清理无用数据
