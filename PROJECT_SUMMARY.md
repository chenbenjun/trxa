## 📱 华为平板安装步骤

### 前提条件
- 华为平板（鸿蒙OS 4.2+）
- WiFi连接（首次安装需要）
- 华为浏览器或Chrome

### 安装流程
1. 访问HTTPS应用地址
2. 等待页面加载完成
3. 点击浏览器菜单 -> "添加到主屏幕"
4. 确认添加
5. 在主屏幕打开"陶然小灶"应用

### 离线使用
- 首次在线访问后，Service Worker会缓存所有资源
- 之后可以关闭网络，离线使用
- 所有功能都可以离线使用（除打印功能）

详细使用指南请参考 `README_HARMONY.md`

## ⚠️ 注意事项

### 必须注意
1. **首次访问必须使用HTTPS**，否则无法安装PWA
2. **首次必须在线访问**，让Service Worker缓存资源
3. **图片大小限制**：每张不超过10M，最多5张/菜品
4. **打印机配置**：必须和华为平板在同一WiFi网络

### 建议配置
1. 横屏使用平板，体验更好
2. 定期备份数据
3. 修改默认管理密码（123456）
4. 使用华为浏览器或Chrome
