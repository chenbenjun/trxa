# 陶然小灶点菜系统 - 快速使用指南

## 🎉 系统已就绪！

**访问地址**：http://localhost:5000

---

## 📱 快速开始

### 第一步：打开系统
在浏览器中打开：**http://localhost:5000**

### 第二步：配置打印机（可选）
1. 点击底部导航栏的 **"管理"** 按钮
2. 点击 **"设置"**
3. 输入打印机IP地址（如：192.168.0.113）
4. 点击"保存"

### 第三步：上传菜品
1. 点击 **"管理"** → **"增加菜品"**
2. 上传菜品图片（最多5张，每张<10MB）
3. 填写菜品信息：
   - 名称
   - 价格
   - 描述
   - 分类
   - 标签（可选）
4. 点击"添加"

### 第四步：开始点菜
1. 点击 **"菜品"** 导航
2. 滑动浏览不同分类
3. 点击菜品卡片查看详情
4. 点击 **"加入购物车"** 或点击 **"+"** 按钮
5. 在购物车中选择规格（忌口、做法等）
6. 点击 **"确认下单"**
7. 点击 **"打印订单"** 发送到厨房

### 第五步：查看进度
1. 点击 **"进度"** 导航
2. 查看所有就餐中的订单
3. 点击订单卡片查看详情
4. 点击 **"加菜"** 继续添加菜品

### 第六步：结账
1. 点击 **"结账"** 导航
2. 找到对应桌号
3. 点击 **"结账"** 按钮
4. 确认后订单完成

## 🎨 自定义主题

### 修改背景
1. **管理** → **"背景"**
2. 上传背景图片
3. 调整：
   - 透明度（0-100%）
   - 模糊度（0-20px）
4. 点击"保存"

### 修改Logo
1. **管理** → **"LOGO"**
2. 上传Logo图片
3. 裁剪调整
4. 调整大小（10-30%）
5. 点击"保存"

## 🔧 后台管理

### 图片管理
- 拖拽排序菜品顺序
- 点击查看大图
- 左右滑动查看多图

### 价格管理
- 直接编辑菜品价格
- 支持小数点（如：28.5）

### 标签管理
- 5个分类：忌口、做法、口味、份量、自定义
- 每个分类可添加自定义标签
- 支持Emoji图标

## 📊 数据说明

### 本地存储
所有数据保存在浏览器本地存储中：
- ✅ 菜品数据
- ✅ 购物车数据
- ✅ 订单数据
- ✅ 背景和Logo
- ✅ 配置信息

### 注意事项
⚠️ **清空浏览器缓存会删除所有数据！**

建议定期导出数据备份。

## ❓ 常见问题

### Q: 页面打不开？
A: 检查服务是否启动：
```bash
curl http://localhost:5000
```

### Q: 图片上传失败？
A: 检查：
- 图片大小（<10MB）
- 图片格式（JPG、PNG、WebP）
- 浏览器存储空间

### Q: 打印机无法连接？
A: 检查：
- 打印机IP是否正确
- 网络连接是否正常
- 打印机是否开启

### Q: 数据丢失？
A: 
- 不要清空浏览器缓存
- 定期导出数据备份
- 使用浏览器的数据导出功能
