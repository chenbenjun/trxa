# 陶然小灶点餐系统 - 华为平板安装使用指南

## 设备信息

- **设备型号**：华为平板（10.8英寸）
- **操作系统**：鸿蒙OS 4.2
- **网络要求**：首次安装需要联网，之后可离线使用

---

## 安装步骤

### 第一步：准备网络环境

⚠️ **重要提示**：首次安装必须连接到互联网，且必须使用HTTPS链接。

1. 打开华为平板的WiFi设置
2. 连接到可用的WiFi网络
3. 确保网络连接正常

### 第二步：打开浏览器

**推荐浏览器**：
- ✅ 华为浏览器（最佳兼容性）
- ✅ Chrome浏览器
- ❌ 微信内置浏览器（不支持PWA安装）

打开浏览器后，访问你的应用地址：
```
https://你的应用地址.com
```

例如：
```
https://你的用户名.github.io/taoran-order-system/
```

或

```
https://taoran-order-system.vercel.app
```

### 第三步：首次访问

1. 首次访问时，浏览器会加载页面
2. 等待页面完全加载完成（可能需要10-30秒）
3. 看到点餐系统界面后，继续下一步

### 第四步：安装PWA到主屏幕

**方法一：使用浏览器菜单（推荐）**

1. 点击浏览器右上角的 `⋮` 或 `⋰` 菜单按钮
2. 在菜单中查找以下选项之一：
   - "添加到主屏幕"
   - "添加到桌面"
   - "安装应用"
   - "Install App"
3. 点击该选项
4. 在弹出的对话框中，确认添加
5. 等待图标出现在主屏幕上

**方法二：使用分享功能**

1. 点击浏览器底部的分享按钮 `⋮` 或 `⋰`
2. 选择"添加到主屏幕"
3. 确认添加

**方法三：长按页面**

1. 在页面空白处长按
2. 选择"添加到主屏幕"

### 第五步：验证安装

1. 回到平板主屏幕
2. 找到"陶然小灶"图标（红色背景，有"陶"字）
3. 点击图标打开应用
4. 应该会像原生应用一样全屏打开（没有浏览器地址栏）

✅ 如果看到全屏界面，说明安装成功！

## 使用说明

### 在线使用

**首次使用**：
1. 确保连接到互联网
2. 打开"陶然小灶"应用
3. 系统会自动缓存所有资源
4. 可以正常点餐、下单、打印

### 离线使用（核心功能）

**首次离线使用**：
1. 至少在线访问过一次（让Service Worker缓存资源）
2. 关闭WiFi和移动数据
3. 打开"陶然小灶"应用
4. 应该可以正常使用所有功能

**离线功能说明**：
- ✅ 菜品浏览（所有菜品已缓存）
- ✅ 购物车管理
- ✅ 订单下单和记录
- ✅ 管理后台（菜品管理、价格管理、背景设置等）
- ✅ 本地数据存储（localStorage）
- ❌ 打印功能（需要连接到本地WiFi网络上的打印机）
- ❌ 图片上传（需要网络传输）

### 数据存储说明

**所有数据都保存在平板本地**，不会上传到云端：

- 📦 菜品列表（包括图片）
- 🛒 购物车数据
- 📝 订单记录
- ⚙️ 系统设置（背景、Logo、打印机配置等）
- 🏷️ 自定义标签

**数据持久化**：
- 数据保存在浏览器的localStorage中
- 关闭应用或重启平板后，数据仍然存在
- 不会因为网络断开而丢失数据

## 功能使用指南

### 1. 点餐流程

1. **选择桌号和人数**
   - 点击左上角的桌号选择器
   - 选择桌号（1-20）
   - 选择就餐人数

2. **浏览菜品**
   - 左侧选择菜品分类（特色、江湖菜、素菜等）
   - 横向滑动查看更多菜品
   - 点击菜品可查看大图

3. **添加到购物车**
   - 点击菜品卡片上的 `+` 按钮
   - 可选择口味和忌口（可选）
   - 点击确认添加

4. **下单**
   - 点击右上角的购物车图标
   - 确认订单信息
   - 选择"下单"或"下单并打印"

### 2. 管理后台

1. 进入管理模式：
   - 点击右上角的 `设置` 图标
   - 输入管理密码（默认：123456）
   - 进入管理后台

2. 管理功能：
   - **图片管理**：修改菜品图片
   - **价格管理**：批量修改菜品价格
   - **标签管理**：自定义忌口标签
   - **增加菜品**：添加新菜品
   - **背景设置**：自定义背景图片
   - **LOGO设置**：自定义LOGO
   - **打印设置**：配置打印机IP

### 3. 订单管理

1. 查看订单：
   - 点击"订单"标签
   - 查看所有活跃订单

2. 结账：
   - 选择订单
   - 点击"结账"按钮
   - 打印消费单
   - 订单自动移至"今日汇总"

3. 加菜：
   - 点击订单
   - 选择"加菜"
   - 添加菜品后下单

### 4. 今日汇总

1. 查看汇总：
   - 点击"报表"标签
   - 查看今日订单统计

2. 清空报表：
   - 点击"清空今日"按钮
   - 确认清空

## 常见问题

### Q1: 无法安装到主屏幕

**原因**：
- 没有使用HTTPS链接
- 浏览器不支持PWA
- manifest.json配置错误

**解决方法**：
1. 确保访问的是HTTPS地址（以https://开头）
2. 使用华为浏览器或Chrome
3. 清除浏览器缓存后重试
4. 检查网络连接

### Q2: 离线无法使用

**原因**：
- Service Worker未正确缓存资源
- 首次未在线访问过

**解决方法**：
1. 连接网络，在线访问一次应用
2. 等待页面完全加载
3. 等待Service Worker缓存完成（约30秒）
4. 断开网络测试

### Q3: 打印功能无法使用

**原因**：
- 打印机未连接到WiFi
- 打印机IP配置错误
- 平板和打印机不在同一网络

**解决方法**：
1. 确保打印机连接到WiFi
2. 进入管理后台 -> 打印设置
3. 配置正确的打印机IP地址
4. 点击"测试连接"验证

### Q4: 数据丢失

**原因**：
- 清除了浏览器数据
- 卸载了应用

**解决方法**：
1. 定期备份重要数据（使用导出功能）
2. 不要清除浏览器缓存
3. 不要卸载应用

### Q5: 应用运行缓慢

**原因**：
- 首次加载需要缓存资源
- 图片文件过大

**解决方法**：
1. 首次使用时耐心等待缓存
2. 在管理后台优化图片大小
3. 使用高质量的WiFi网络

### Q6: 图片不显示

**原因**：
- 图片未缓存
- 文件损坏

**解决方法**：
1. 在线访问一次，让图片缓存
2. 在管理后台重新上传图片
3. 检查图片大小是否超过限制（每张≤10M）

## 性能优化建议

### 1. 图片优化

- 使用JPG格式（比PNG小）
- 控制图片尺寸（建议500x500像素）
- 每张图片不超过10M

### 2. 缓存管理

- Service Worker会自动缓存资源
- 缓存有效期为1年
- 不需要手动清理缓存

### 3. 存储空间

- 本地数据占用空间很小（约50-100MB）
- 主要空间占用：菜品图片
- 建议定期清理不用的菜品图片

## 更新应用

当开发人员更新了应用后：

1. 连接到网络
2. 打开"陶然小灶"应用
3. Service Worker会自动检测更新
4. 首次打开可能需要重新缓存（约30秒）
5. 关闭应用重新打开即可使用新版本

**手动强制更新**：
1. 进入浏览器设置
2. 清除浏览器缓存
3. 重新访问应用地址
4. 重新安装到主屏幕

---

## 备份与恢复

### 备份数据

1. 进入管理后台
2. 找到"导出数据"功能
3. 导出所有数据为JSON文件
4. 保存到云端或电脑

### 恢复数据

1. 进入管理后台
2. 找到"导入数据"功能
3. 选择之前导出的JSON文件
4. 恢复数据

1. **设置复杂的管理密码**
   - 修改默认密码（123456）
   - 使用字母、数字、符号组合

2. **定期备份数据**
   - 每周备份一次订单数据
   - 保存到云端或外部存储

3. **保护设备安全**
   - 设置平板开机密码
   - 不要让陌生人操作管理后台
