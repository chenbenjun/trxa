# 陶然小灶 - 智能点餐系统

> **纯离线单机版** - 无需服务器、无需联网、即开即用

一个专为餐厅设计的智能点餐系统，采用中国传统美学风格，支持iPad、手机等多种设备访问。本系统采用纯前端架构，所有数据本地存储，完全离线可用。

## ✨ 核心特性

### 🎨 界面设计
- **中国传统美学**：朱红色、金色、墨黑色等传统配色
- **响应式布局**：完美适配手机、平板、电脑屏幕
- **自定义品牌**：支持上传自定义背景图片和Logo
- **现代化UI**：基于shadcn/ui组件库，界面简洁优雅

### 🍽️ 菜品管理
- **分类浏览**：特色菜、主菜、凉菜、热菜、饮品等多个分类
- **图片展示**：每个菜品支持最多5张高清图片
- **详细信息**：菜品名称、价格、详细介绍
- **标签系统**：支持口味、忌口、做法、份量等标签管理

### 🛒 购物车功能
- **实时管理**：添加、删除、修改菜品数量
- **智能提醒**：支持口味选择、忌口备注
- **自动保存**：购物车数据自动保存到本地存储
- **加菜模式**：支持对已下单桌号进行加菜

### 📄 订单管理
- **实时订单**：查看所有进行中的订单
- **订单详情**：显示订单号、桌号、人数、菜品明细
- **自动编号**：订单号按日期自动生成（年月日+序号）
- **快速结账**：一键完成订单并记录

### 🖨️ 打印功能
- **本地打印**：无需网络配置，直接调用系统打印机
- **测试打印**：快速测试打印机是否正常工作
- **订单打印**：支持打印消费明细单
- **格式优化**：58mm热敏纸格式，清晰易读

### 💾 数据管理
- **纯离线存储**：所有数据存储在浏览器localStorage中
- **自动备份**：支持数据导出备份
- **图片Base64**：图片以Base64格式本地存储
- **数据持久化**：刷新页面数据不丢失

## 🚀 快速开始

### 系统要求
- 任何支持现代浏览器的设备（iPad、手机、电脑）
- 推荐使用Chrome、Safari、Edge浏览器

**方式一：在线部署（开发测试）**

```bash
# 安装依赖
pnpm install

# 启动开发服务器
coze dev
```

访问：http://localhost:5000

**方式二：纯离线使用（生产环境）**

```bash
# 构建生产版本
coze build

# 将生成的 .next、public、node_modules 等文件
# 复制到任意目录，双击 index.html 或使用静态服务器打开
```

**方式三：直接使用构建产物**

1. 运行 `coze build` 构建项目
2. 将生成的文件复制到目标设备
3. 使用任何静态文件服务器打开（如 Python HTTP Server）
4. 无需任何网络连接即可使用

## 🎯 使用指南

### 点餐流程

1. **选择桌号**：在左上角选择桌号和人数
2. **浏览菜品**：在主界面浏览各分类菜品
3. **添加菜品**：点击菜品卡片添加到购物车
4. **设置标签**：在购物车中设置口味、忌口等标签
5. **下单确认**：点击"下单"按钮，可选择打印或不打印
6. **查看进度**：在"进度"页查看所有进行中的订单

### 管理功能

点击右上角"管理"按钮进入管理模式，包含以下功能：

1. **图片管理**：查看和编辑菜品图片
2. **价格管理**：批量修改菜品价格
3. **标签管理**：管理口味、忌口等标签
4. **增加菜品**：上传新菜品图片和信息
5. **背景设置**：自定义页面背景
6. **Logo设置**：上传餐厅Logo
7. **打印设置**：测试本地打印功能

### 打印功能

本系统采用本地打印模式：

1. 在管理页面进入"打印设置"
2. 点击"测试打印"验证打印机是否正常
3. 下单时选择"打印"按钮即可打印订单
4. 系统会弹出打印预览窗口，确认后执行打印

**支持的打印机类型：**
- 热敏打印机（USB或网络连接）
- 普通打印机（A4或58mm热敏纸）
- PDF虚拟打印机

## 📝 注意事项

### 存储限制

- 浏览器localStorage通常限制在5-10MB
- 图片会自动压缩，但建议控制图片大小
- 建议定期导出数据备份

### 打印设置

- 首次打印时需要选择打印机
- 确保打印机已正确连接到设备
- 检查浏览器是否允许弹窗

### 数据备份

```typescript
// 在浏览器控制台执行导出
exportData(); // 下载JSON备份文件

// 数据保存在浏览器localStorage中
// 键名包括：taoran-dishes、taoran-cart、taoran-orders等
```

### 清除数据

如需清除所有数据：

```javascript
// 在浏览器控制台执行
localStorage.clear();
location.reload();
```

## 🐛 常见问题

**Q: 图片上传后不显示？**
A: 检查图片大小是否超过10MB，建议使用压缩后的图片。

**Q: 打印时没有反应？**
A: 检查浏览器是否允许弹窗，确保打印机已正确连接。

**Q: 刷新页面后数据丢失？**
A: 检查浏览器是否禁用了localStorage，或是否开启了隐私模式。

**Q: 如何在iPad上离线使用？**
A: 将构建后的文件保存到iPad，使用文件管理器或其他应用打开即可。

### 本地存储

使用 `src/lib/storage.ts` 中的工具函数：

```typescript
import { saveDishes, loadDishes } from '@/lib/storage';

// 保存数据
saveDishes(dishes);

// 加载数据
const dishes = loadDishes();
```

### 图片处理

使用 `src/lib/image-utils.ts` 中的工具函数：

```typescript
import { fileToBase64, compressImages } from '@/lib/image-utils';

// 转换为Base64
const base64 = await fileToBase64(file);

// 压缩图片
const compressed = await compressImages([base64]);
```

### 打印功能

使用 `src/lib/print.ts` 中的工具函数：

```typescript
import { printOrder, printTest } from '@/lib/print';

// 打印订单
printOrder(order);

// 测试打印
printTest();
```
