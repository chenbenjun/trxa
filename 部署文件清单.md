## 🚀 使用建议

### 如果你想直接使用部署脚本

1. **选择适合你的脚本：**
   - Windows：`deploy-github-pages.bat`
   - Mac/Linux：`deploy-github-pages.sh`

2. **准备：**
   - GitHub账号
   - Personal Access Token

3. **执行脚本**
   - 按提示操作

4. **完成！**

### 如果你想手动部署

1. **解压压缩包**
   ```
   tar -xzf taoran-point-system-pwa.tar.gz
   ```

2. **上传到GitHub**
   - 创建仓库
   - 推送文件

3. **配置GitHub Pages**
   - Settings → Pages
   - Deploy from a branch
